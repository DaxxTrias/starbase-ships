## Comparison

|   | Bugger | Bugger X |
|---|---|---|
| Thrusters | T2 | T3 |
| Top Speed (empty) | 100m/s | 120m/s |
| Generators | T3 | T3 |
| Fuel Chambers | T3 | T3 |
| Enhancers | T2 | T3 |
| Power Generation | 72,000eps | 81,000eps |
| Mining Time | 90mins | 100mins |
